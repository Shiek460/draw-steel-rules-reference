- Minions are weak, meant to die fast.
- Shared Stamina Pool -> Stamina x # of minions
- Some have **Captain Benefits**, denoted by *With Captain:* section.
# Action Economy
- Can only move/action, move/maneuver, or move/move
- Can make opportunity attacks
- **One Roll** for each squad. Use same result for however many minions use ability
	- No more than **3** signatures directed at one PC. Any more attackers, use *free strike* damage
- If minion squad crits, all can use same action again
# Captains
- Any non minion with a shared language
- If captain falls, a new one can be selected at top of the round, for free
